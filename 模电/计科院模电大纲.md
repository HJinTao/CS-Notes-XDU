## 第三章 放大电路频率特性（了解即可）

- 频率特性的概念

    - 由于**电容**存在，不同频率呈现的阻抗不一样，放大倍数与频率的关系称为幅频关系，相位与频率的关系称为相频关系，

- 线性失真

    - 由于放大器放大的信号经常是由不同频率的信号组合而成，因此会产生幅频失真和相频失真，统称为 **线性失真**
    - 输入信号为单一信号时，一定无线性失真

- 上限频率$f_l$，下限频率$f_h$，通频带宽度$f_{BW}$

    通常定义放大倍数下降到中频区放大倍数的$\frac{1}{\sqrt{2}}$的频率为截止频率，也就是$A_u=\frac{1}{\sqrt{2}}A_{um}$

    低频段的截止频率为下限频率，高频段对应的为上限频率，两者差为频带宽度

- 影响频率特性的因素（可以记忆电阻和电容串联模型，结合电容的阻抗公式）
    - 低频段的频率响应
        - 耦合电容
        - 旁路电容
    - 高频段的频率响应
        - 三极管的极间电容
        - 电路分布电容

        - ~~P沟道**增强型**场效应管~~
        - ~~P沟道**耗尽型**场效应管~~

- 符号、转移特性曲线与输出特性

    具体示意图见课本P88

    - $根据输出特性i_D - U_{GS}的 U_{GS}范围可以判断是哪种场效应管（一般是N沟道）$
        - 小于零：结型场效应管
        - 大于零：增强型场效应管
        - 有正有负：耗尽型场效应管
    - N沟道 VS P沟道

        - 耗尽型和结型场效应管

        - 图示：实线


## 第五章 负反馈放大电路

- 反馈类型的判定
    - 输入端：串联 or 并联，看反馈电阻的另一端是否与$U_i$一致，若一致则并联
    - 输出端：电压 or 电流，短路$U_o$，看反馈量是否马上变0，若是则电压
    - 交流 or 直流
    - 正反馈 or  负反馈
- 四种基本组态的负反馈放大器
    - 单位判断
        - 放大倍数
        - 反馈系数
        - 示例（以串联电流为例）
            1. 放大倍数：串联代表输入，也就是$U_i,U_f$，电流代表输出，也就是$I_o$，所以放大倍数为$\frac{I_o}{U_i}$
            2. 反馈系数：同上，反馈系数为 $\frac{U_f}{I_o}$

    - XX控制XX源
        - 示例：电流控制电压源
        - 第一个电流/电压代表 输入，分别对应并联/串联
        - 第二个电流/电压不需要变
        - 所以电流控制电压源的等效为：并联电压负反馈电路

    - 输入信号源
        - 电流源：并联
        - 电压源：串联

- 负反馈对放大器性能的影响
    - 基本公式
        1. $A_{f} = \frac{A}{1+AF}$
        2. $\frac{\Delta A_f}{A_f} = \frac{1}{1+AF}  \cdot \frac{\Delta A}{A}$，系数称为反馈系数，很重要
    - 输入电阻
    - 输出电阻
    - 上限频率
    - 下限频率




## 第六章 集成运算放大器

- 零点：当交流输入信号为0时，放大器的输出电压值较做放大器的零点
- 零点漂移
    - 放大器的零点随时间变化，主要是由晶体管参数随温度变化引起
    - 计量：$\frac{U_{o0}}{A_{u}}$，也就是除以放大系数，该计量数越大，零漂越大
- 差动放大器主要利用对称特性以及 Re的负反馈来抑制零漂
- 长尾差动放大器
- 各种信号
    - 差模输入信号：$U_{id}=U_{i1} - U_{i2}$ 
    - 差模信号：$U_{id1} = -U_{id2} = \frac{1}{2}U_{id}$
    - 共模输入信号：$U_{ic} = \frac{U_{i1}+U_{i2}}{2}$
    - 共模信号：$U_c = U_{ic}$
    - 任意信号：可以拆分成共模信号+差模信号
- 差动放大器的指标
    - $A_{ud}=\frac{U_{od}}{U_{id}}$
        - 具体电路图计算参数：转换成单边放大参数
        - 应用题数值计算参数：注意正负，定性分析下正负再决定是正是负，若没给图则默认正
    - $A_{uc}=\frac{U_{oc}}{U_{ic}}$
        - 具体电路图计算参数：一般是0
        - 应用题数值计算参数：注意正负，定性分析下正负再决定是正是负，若没给图则默认正
    - $CMRR =| \frac{A_{ud}}{A_{uc}}| $
        - 越大对共模信号的抑制能力越强
    - $CMR = 20 log| \frac{A_{ud}}{A_{uc}}|$
    - $U_o = A_{ud}U_{id} + A_{uc}U_{ic}$
        - 注意各项的正负
        - $U_{id}$直接默认公式



## 第七章 集成运算放大器的应用

- 虚短 VS 虚断

    - 虚短：$U_+ = U_-$，是指在分析运算放大器处于线性状态时，可把两输入端视为等电位（非线性运用不成立）
    - 虚断：$I_+ = I_- = 0$，是指理想运放的输入电流趋于0（始终成立）

- 线性运用 VS 非线性运用

    - 线性运用：理想运放在外部引入深度负反馈后，满足虚短和虚断
    - 非线性运用：理想运放开环工作或者引入正反馈后，满足虚断，不满足虚短

- 平衡电阻计算

- 运算电路

    1. 反相比例运算电路：虚地，共模输入信号约等于0，$CMRR$要求不高
    2. 同相比例运算电路：输入电阻无穷大，共模输入信号高，$CMRR$要求高
    3. 差动比例运算电路：由叠加定理从1,2推导即可
    4. 反相求和电路：有叠加定理从1推导即可
    5. 同相求和电路：注意一般推导不用叠加定理，而是运用虚短+虚断，注意$Uo$的计算
    6. ~~代数求和电路~~
    7. 积分电路：$R_f$的位置换成$C$，注意$i = C\frac{du}{dt}$，注意电容充满电后的波形图画法
    8. 微分电路：$C$和$R$的位置互换即可

- 滤波电路

    - 通用模型：$R和C串联，根据不同的频率选择分压大的R或者C输出$
    - 分类
        1. 低通滤波电路：输出$C$
        2. 高通滤波电路：输出$R$
        3. 带通滤波电路：低通滤波和高通滤波串联，可以画数轴图
        4. 带阻滤波电路：并联，也可以画数轴图

    - 无源滤波器：也就是通用模型
    - 有源滤波器：一般都是采用同相比例运算电路，$U_+$的选择基于通用模型

- 电压比较器

    - 基本概念
        - 运放工作在非线性区
        - 比较器的阈值$U_{th}$计算由$U_+ = U_-$推导而出，根据阈值的个数分为单限和滞回
        - 组态：输入信号在 **-**端则为反相，反之则为同相
    - 分类
        1. 简单电压比较器（单限电压比较器）
        2. 滞回比较器
            - 根据不同的$U_o$有不同的阈值，有两个阈值
            - 通过运放电路引入正反馈获得
            - 画传输特性图时比较难判断的是初值，初值确定了就可以确定阈值，阈值确定了就简单了
    - 传输特性图
        - 注意稳压二极管是否有导通电压，若有则要加上导通电压

- 分析

    - 灵活运用反相比例运算电路和同相比例运算电路公式
    - 若无法运用上述公式，则列出虚短和虚断公式，同时结合电压电流规律

- 设计

    - 代数求和电路：多级反相比例运算电路串联
    - 其他设计电路都可以基于多级反相比例运算电路获得

## 第九章 低频功率放大电路（了解即可）

- 按功率放大电路中晶体管在一个信号周期内导通时间的不同，分为：甲类、乙类和甲乙类。
- 各自的特点
    1. 甲类：导通角$2\pi$，工作状态失真小，静态电流大，管耗大，效率低。
    2. 乙类：导通角$\pi$，工作状态失真大，静态电流为零 ，管耗小，效率高
    3. 甲乙类：导通角$\pi <  \theta < 2\pi$，工作状态失真大， 静态电流小 ，管耗小，效率较高。
- 交越失真
    - 概念：输入信号很小时，达不到三极管的开启电压，三极管不导电。因此在输入信号正、负半周交替过零处，没有输出信号，使输出信号产生非线性失真，这个失真称为交越失真。
    - 属于非线性失真

## 第十章 直流电源（了解即可）

- 概述

    - 把交流电源变为直流电源需要经历：$u1变压u2，整流，滤波，稳压$

- 整流：利用二极管的单向导电特性，将正负交替的交流电压变换成单方向的直流脉动电压。

    - 单相半波
    - **单相桥式**（重要）
        - 图示画法
        - 输出电压平均值 $U_o = 0.9U_2（都是有效值），注意是d(wt)$
        - 输出电流平均值$I_o$
        - 二极管平均电流$I_D = \frac{1}{2}I_o ，注意有\frac{1}{2}$
        - 二极管最大承受电压$\sqrt{2}U_2$

- 滤波：整流电路的输出都含有较大的脉动成分，滤波电路可以降低输出电压中的脉动成分，还可以尽量地保留其中的直流成分，使输出电压更加平滑，接近于直流电压。

    - 电容滤波（最常用）

        - 原理
            1. $电容阻抗X_c = \frac{1}{jwc}$
            2. 对于输出电压中的直流分量，电容**C**相当于开路
            3. 对输出电压中的交流分量，只要**C**的容量取得足够大，$X_c$可做得很小，**C**就相当于短路
            4. 这样，输出电压中的直流分量流过$R_L$，而交流分量则由电容旁路（即：进行了滤波）
        - 输出电压平均值 $U_o = 1.2U_2（都是有效值）由于进行了滤波，积分平均值变大，大于0.9U_2$
        - 滤波电容最大承受电压$\sqrt{2}U_2$
        - 注意点
            1. 选择大容量的电容 **C**，$R_L C \ge(3 ～5) \cdot \frac{T}{2}$

        

- 稳压：在整流、滤波环节后面加接稳压电路，能使直流电源的输出电压基本上不随交流电网电压波动或负载变化而变化

    - 稳压管电路
    - 串联型稳压电路
        - **组成：取样网络、基准电压、比较放大器、调整元件。    重要！必考**
        - 工作过程
            1. 取样网络：由电阻，电位器组成，作用是取出输出电压的变化量
            2. 基准电压：向比较放大器提供一个基准电压
            3. 比较放大器：对输出电压变化量进行放大从而对调整元件进行负反馈调节
            4. 调整原件：通常是三极管，管压降$U_{CE}$的自动调节使输出电压稳定
    - 集成稳压器
        - 型号解读
            1. CW78XXX 系列（正电源）
            2. CW79XXX 系列（负电源）
            3. 输出电压：由最后两位决定
            4. 输出电流：由倒数第三位决定，空代表1.5A，M代表500mA，L代表100mA
            5. 例子
                1. CW7805   输出 5 V，最大电流 1.5 A
                2. CW79M05   输出 － 5 V，最大电流 0.5 A
                3. CW78L05   输出 5 V，最大电流 0.1 A
        - 基本电路
