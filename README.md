# CS-Notes-XDU

适用于西安电子科技大学计算机科学与技术专业 2024 级及以后的专业课笔记。

## 现有内容

- **操作系统**：待完善。
- **计组**：期中与期末复习文档+各章例题图片
- **模电**：计科院模电大纲 + PPT + 参考试卷

笔记以 Markdown 编写，配套题目等图片资源存放于各课程的 `img` 子目录或对应章节文件夹内。

## 使用

克隆仓库后，用支持 Markdown 的编辑器打开即可，推荐 [Obsidian](https://obsidian.md)、[Typora](https://typora.io) 。图片链接基于相对路径，直接浏览即可显示。

```bash
git clone https://github.com/HJinTao/CS-Notes-XDU.git
```
